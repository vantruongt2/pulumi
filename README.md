# pulumi
This repository contains infrastructure as code (IaC) configurations using Pulumi to manage cloud resources programmatically. The project is structured to support multiple services, each with its own deployment logic.

## Python Project Structure
This section outlines the structure of the Python-based Pulumi project. It includes various modules for different services and a factory pattern to manage service deployments.
Go to [Python Project Structure](./python/README.md) for more details.
